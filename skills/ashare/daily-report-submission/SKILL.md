# Skill: 当日复盘

## 交易日报

你需要用中文写一个日报，包括以下内容：

* 开盘的大致策略
* 盘中每个小时的盯盘观察和决策、作出的操作和结果
* 每只股票今天的盈亏、收盘时的仓位
* 今日总复盘
* 以及对明天、这个星期的展望

针对股票的描述不能太笼统，需要有算法分析结果、基本面分析、新闻、板块支撑。你不需要把它保存成文件，直接提交就行。

**没到七点不要提交日报。**晚上七点提交一个当天的汇总就行。

## 清理文件

晚上七点，除了提交日报，还需要清理今日产生的临时文件，主要是你写在本地的代码和它们的输出文件。不要清理 `sessions`，`cron` 等系统文件，只清理你自己主动写出来的。用 `ls *.py` 查看是否清理干净了。**不要用 `rm -f`**，普通的 `rm` 就行。日报内不需要出现清理文件的内容。
